HTML Fundamentals

This activity is the starting point of my web development journey. I learned how to build the backbone of a webpage using basic HTML tags, focusing on structure rather than design. By creating a multi-section webpage entirely from scratch with raw HTML, I was able to see how headings, paragraphs, and sections create a logical flow that makes content easier to understand and manage.

At the same time, I applied the lessons I had learned from weeks 1 through 7, which helped me improve the overall quality of my webpage and make it more effective and visually meaningful.

W04 DOM

JavaScript & DOM Basics

At this stage, I learned how to make my webpage interactive by using JavaScript together with the DOM. I built a simple Task Manager where users could add, mark, or delete tasks, and a Contact Form that validates inputs before submission. These features showed me how JavaScript can control page elements directly, turning static content into something functional and user-driven.

Even though the activity was my first step into DOM manipulation, it became an upgraded version of my earlier outputs; proving how logic and interactivity can enhance the overall user experience.

W06 APIs

API & Client Storage

At this stage, I explored how APIs and client storage can extend the functionality of a webpage. I worked with the GitHub API to fetch user data and learned how to store information locally, which showed me how web applications can connect with external services while keeping user preferences on the client side. I also integrated the OpenWeather API to display real-time weather data such as temperature, humidity, and conditions in a clean, styled interface.

This activity helped me understand how modern web systems rely on APIs for dynamic content and how client storage ensures a smoother, more personalized user experience. It became an upgraded version of my earlier outputs, combining interactivity, external data, and persistence.
