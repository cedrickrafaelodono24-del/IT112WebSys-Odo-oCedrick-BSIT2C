Procedure
01

Installing Visual Studio Code

I Download and set up VS Code as my primary development environment, then i install the essential web development extensions.

1I Go to code.visualstudio.com and click Download
2I Select the correct version for my OS (Windows / macOS / Linux)
3Running the installer and following the on-screen instructions
4Launch VS Code and explore the interface
5I Install the following extensions from the Extensions panel
HTML Snippets
Live Server
Prettier — Code Formatter
02

Installing Sublime Text

Setting up Sublime Text as my secondary editor and get familiar with its interface and workflow.

1I Go to sublimetext.com and click Download
2Choose the appropriate version for my operating system
3Running the installer and following the on-screen instructions
4Launch Sublime Text and explore the interface
03

Testing in my VS Code

Creating my first HTML file inside VS Code and opening it in the browser to verify everything is working correctly.

1I Open VS Code and create a new folder named Week1_Setup on my desktop
2Inside the folder, I create a new file named index.html
3I Copy and paste the code below into the file
4I Save the file and open it in my browser to verify the output
index.html 
<!DOCTYPE html>
<html>
<head>
    <title>Lab 1 Test</title>
</head>
<body>
    <h1>Hello, Web Technologies!</h1>
    <p>This is a test file created using Visual Studio Code.</p>
</body>
</html>
04

Repeating in Sublime Text

Open the same file in Sublime Text, make a small change, and verify the updated output shows correctly in the browser.

1I Open Sublime Text and navigate to the Week1_Setup folder
2I Open the index.html file you created
3I Change the <h1> text to "Welcome to Sublime Test!"
4I Save the file and refresh the browser to verify the change
Expected Output

Both IDEs should produce a functional HTML page. The browser should render the heading and paragraph correctly in each case.

VS Code Output
Hello, Web Technologies!
This is a test file created using Visual Studio Code.
Sublime Text Output
Welcome to Sublime Test!
This is a test file created using Visual Studio Code.
